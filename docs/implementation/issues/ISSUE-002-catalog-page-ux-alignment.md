# ISSUE-002 — Catalog page UX alignment with homepage

## Problem
Category/catalog pages diverge from homepage visual system and include broken/awkward slider behavior. Search sidebar parity is also inconsistent.

## Scope
- Align category/catalog product presentation with homepage card system and spacing conventions.
- Replace broken slider usage with promotional slider sections that work reliably.
- Add search sidebar in pages currently using alternate components where requested.

## Acceptance Criteria
- [ ] Category pages use the same card visual language as homepage product system.
- [ ] Broken slider interaction is removed/replaced by working promotional sliders.
- [ ] Search sidebar is visible and functional on targeted catalog pages.
- [ ] No extra pages, modals, or new theme primitives are introduced.

## Validation
- Manual UI verification in desktop and mobile breakpoints.
- Live check on `https://blue-meadow-00fcb8810.4.azurestaticapps.net/category` and `/category/[slug]` pages.
